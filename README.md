# Sistema de Cadastro IFTO - POO PHP

Este projeto é uma atividade acadêmica feita pelo aluno Gustavo Romão, que implementa um sistema de cadastro de frequentadores do campus utilizando **Programação Orientada a Objetos (POO)** em PHP.

## Funcionalidades
- Cadastro de 4 tipos de perfis: **Estudante, Professor, Servidor e Visitante**.
- Lógica de herança:
  - `Estudante`: Vinculado a um curso.
  - `Professor` e `Servidor`: Possuem função e salário.
  - `Visitante`: Cadastro simplificado para controle.

## Como rodar o projeto
1. Clone este repositório.
2. Certifique-se de ter um servidor PHP local (XAMPP).
3. Coloque os arquivos na pasta do servidor.
4. Acesse via navegador (ex: `http://localhost/Cadastro/index.php`).
